AdGuard Content Blocker est une application qui bloquera toutes les pubs <b>seulement dans les navigateurs Yandex Browser et Samsung Internet browser</b> sans nécessiter un appareil rooté. AdGuard Content Blocker <b>ne bloque pas</b> les pubs dans les autres applications. Personnalisez l'application en fonction de vos besoins - choisissez parmi l'une des 20 listes de règles de filtrage générales et spécifiques à votre langue disponibles.

L'application est open source et peut être trouvée sur GitHub :
https://github.com/AdguardTeam/ContentBlocker
